# 🔄 Queue using Singly Linked List (C)

## 📌 Overview

This project implements a **Queue (FIFO - First In First Out)** using a **Singly Linked List** in C with a menu-driven interface.

---

## 🚀 Features

* **Insert (Enqueue)** → Add element at rear
* **Remove (Dequeue)** → Remove element from front
* **Display** → Show all queue elements
* Handles **Overflow** and **Underflow**

---

## 🧠 Concepts

* Linked List (SLL)
* Pointers (Front & Rear)
* Dynamic memory (`malloc`, `free`)

---

## ▶️ Run

```bash id="runq1"
gcc queue_sll.c -o queue
./queue
```

---

## 🖥️ Menu

```bash id="menuq1"
1. Insert
2. Remove
3. Display
```

---

## ⚠️ Notes

* `front` → first element
* `rear` → last element
* Underflow if queue is empty
* Overflow if memory allocation fails
